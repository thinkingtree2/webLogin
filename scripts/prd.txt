# Overview
일반사용자 로그인 및 관리자 로그인 페이지를 제공합니다. 가장 단순한 웹사이트입니다.
이 웹사이트는 다른 웹사이트 개발 시 로그인 기능을 쉽게 사용하기 위한 기초 프로젝트로 사용됩니다.

# Core Features
1. 일반 사용자 회원가입 및 로그인
   - 사용자는 계정을 생성하고 로그인할 수 있습니다.
   - JWT 기반 인증을 사용합니다.
2. 관리자 페이지
   - 관리자는 별도의 로그인 페이지를 통해 접근합니다.
   - 대시보드를 통한 통합 관리 기능 제공
3. 반응형 웹
   - 모든 기기에서 최적화된 UI 제공
4. RESTful API
   - FastAPI 기반의 백엔드에서 RESTful API 제공
5. 데이터베이스 연동
   - PostgreSQL 사용

# User Experience
## 사용자 유형
- 일반 사용자: 회원가입, 로그인, 서비스 이용
- 관리자: 관리자 로그인, 대시보드 접근 및 관리 기능 사용

## 주요 사용자 플로우
- 회원가입 → 로그인 → 서비스 이용
- 관리자 로그인 → 대시보드 접근 → 관리 기능 수행

## UI/UX 고려사항
- ShadCN UI, Tailwind CSS를 활용한 일관된 디자인
- 반응형 웹 구현

# Development Roadmap
## MVP 요구사항
- 회원가입/로그인 기능 (일반 사용자)
- 관리자 로그인 및 대시보드 접근
- 기본 RESTful API 구현
- 반응형 웹 UI
- PostgreSQL 연동 및 기본 데이터 모델 설계


# Logical Dependency Chain
1. 데이터베이스 및 기본 데이터 모델 설계
2. 백엔드(FastAPI) 기본 구조 및 RESTful API 구현
3. 인증(JWT) 로직 구현
4. 프론트엔드(React) 기본 구조 및 회원가입/로그인 UI 구현
5. 관리자 페이지 및 대시보드 구현
6. 반응형 웹 및 UI/UX 개선

# Risks and Mitigations
- 기술적 도전: FastAPI, JWT, PostgreSQL 연동 경험 부족 → 공식 문서 및 예제 참고, 단계별 개발
- 단위테스트 및 통합테스트 단계적 개발
- MVP 정의: 최소 기능에 집중, 이후 점진적 개선
- 리소스: 단순 구조로 시작, 점진적 확장
