# Overview
일반사용자 로그인 및 관리자 로그인 페이지를 제공합니다. 가장 단순한 웹사이트 입니다.
일반사용자에게 보여주는 페이지와 관리자 페이지가 다르게 제공됩니다.
이 웹사이트는 다른 웹사이트를 개발시 로그인 기능을 쉽게 사용하기 위한 기초 프로젝트로 사용됩니다. 


# User Experience
## 사용자 유형
1. 일반 사용자
   - 회원 가입

2. 관리자
   - 모든 관리 기능 접근 가능
   - 대시보드를 통한 통합 관리
